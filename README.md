# Case Converter

Convert text between 12 cases: UPPER, lower, camelCase, PascalCase, snake_case, kebab-case, and more.

**[Try it live →](https://theluckystrike.github.io/bln-case-converter/)**

## Features

- UPPERCASE and lowercase
- Title Case and Sentence case
- camelCase and PascalCase
- snake_case and kebab-case
- CONSTANT_CASE and dot.case
- One-click copy to clipboard
- Real-time conversion

## Usage

Open the tool in your browser — no installation required. Paste or type your text and get instant results.

## Tech Stack

- Pure HTML, CSS, JavaScript
- No dependencies or build step
- Hosted on GitHub Pages
- Works offline after first load
